================================================
Top Level Howdy API
================================================
This document describes the top level API, which defines the location of the Plex_ user configuration directory in a variable, ``baseConfDir = ~/.config/howdy``, and the resources directory in which universal resources (such as icons, fonts, style sheets, and settings for Google OAuth2 authentication) live.
