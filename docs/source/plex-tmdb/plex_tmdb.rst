================================================
TMDB Functionality
================================================

These command line executables, graphical user interfaces, and APIs are used to download new movies, and search and manage your Plex movie library. This documentation is organized into sections on :ref:`get_mov_tor.py`, CLI used to search and download movies; :ref:`plex_tmdb_totgui.py`, the GUI used to manage and search through your Plex movie library, and to search and download new movies; and finally on the :ref:`low level API <PlexTMDB API>`.
