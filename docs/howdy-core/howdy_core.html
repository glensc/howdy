4. Howdy Core Functionality¶

These command line executables, graphical user interfaces, and APIs form the core functionality in Howdy. The API lives in howdy.core. These tools form the infrastructural basis of higher level tools to manage television shows, movies, music, and email.

This documentation is organized into sections on command line tools, GUI tools, low level API, and finally the TODO.
